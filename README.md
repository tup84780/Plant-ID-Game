# Plant ID Challenge

Plant ID Challenge is a browser-based guessing game built with plain HTML, CSS, and vanilla JavaScript. Players receive three clues about a plant species and have up to three tries to identify it.

## Features

- Two game modes: multiple choice and open ended
- Three clues per round
- Three attempts before losing
- Accepts either the common name or scientific name as a correct answer in open ended mode
- Reveals a picture of the correct plant after a win or after three incorrect guesses
- Improved layout with a clearer game panel, status area, and answer/reveal sections
- Cleaner, more readable game logic with dedicated reveal handling
- Works by opening `index.html` directly in a browser

## How to Run

1. Open the project folder.
2. Double-click `index.html`, or right-click it and open it in any modern browser.

No installation, build step, or dependencies are required.

## Files

- `index.html` - the page structure and game interface
- `style.css` - the app styling and layout
- `script.js` - the game data and logic

## How to Play

1. Choose `Multiple Choice` or `Open Ended`.
2. Read the three clues shown on screen.
3. Submit your guess.
4. If you guess correctly, the game reveals the plant image right away.
5. If you miss three times, the correct answer and plant image are revealed.
6. Click `New Plant` to start another round.
